This plugin allows you to use the exclusive Pydio hosted service, FileGateway. By shortening AND proxying your public links to your servers, FileGateway allows to you securely share resources with external users via public links without the risk of revealing your Pydio server URL.

To be able to use this service, please buy a subscription on Pydio.com website and grab the API Keys / API Secret that will be available in your account to configure the plugin. You can choose on which region you wish your links to be "hosted", either in the US (filesend.cc) or in Europe (yoursha.re).
